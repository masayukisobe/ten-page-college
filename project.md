# 10ページ大学 project

## 企画の核

10ページ大学は、AI時代の大人の学び直しシリーズである。

ブランドコピーは、**「知ってる言葉を、説明できる教養へ。」**

読者がニュース、仕事、日常会話、SNSで見聞きしている言葉を、仕組みまで少し説明できる状態へ近づける。専門書や長い講義へ入る前に、入口、見取り図、問い方を渡す。

本文の価値は、読者の見え方を変えることにある。

読後に目指す感覚:

> 名前だけは知っていたけど、そういう仕組みだったのか。ちょっと面白い。

## 読者像

想定読者は、仕事や生活の段取りはできるが、専門的な抽象化や数式には慣れていない大人である。

読者は、AI、半導体、クラウド、文字コード、圧縮、OS、ネットワークのような言葉を、見聞きした経験がある。

本文では、読者の普段の経験、画面で見ているもの、手元で起きる操作から入り、その言葉の裏側を説明する足場を作る。

## 商品としての形

1テーマにつき、本文10ページ程度の電子コンテンツを作る。

「10ページ」は、学びの最小単位を示す編集思想である。

標準構成:

1. はじめに
2. 本文10ページ
3. 重要用語
4. 理解確認クイズ
5. ChatGPT深掘りプロンプト
6. 次に検索するキーワード

本文ページは、図や例示枠を除いた本文だけで平均50行程度、約1000字程度を目安にする。図は1ページ4-5点を標準にする。

## 執筆方針

本文は、ですます調で書く。

読者を子ども扱いせず、知識の入口にいる大人として扱う。

専門語は、必要な場面で使う。ページタイトルにも、本文の説明トーンに合う専門語を使ってよい。目次に少し硬い言葉が並び、本文を読むと説明できるようになることが、ブランドコピーとつながる。

UTF-8、RGB、JPEG、圧縮、コーデックのように、読者が普段目にする語の仕組みを説明できる場合は積極的に使う。名前を出したら、その語が何の約束か、何を変換しているのかを本文で一口ぶん説明する。

本文の面白さは、具体例、順番、比喩、図、短い回収で作る。
