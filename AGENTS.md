# AGENTS.md

## 基本方針

- ユーザーへの返信、進捗報告、質問、最終報告は日本語で行う。
- このリポジトリでシェルコマンドを実行するときは、原則として `rtk` を前置する。
- Node 系プロジェクトでは、明示的に別指定がない限り `pnpm` を優先する。
- 文章を書くときは、その文章の読み手が誰で、いつ、何のために読むかを先に考え、その読み手が持つ文脈を前提にする。

## 否定文の扱い

- 記事本文、方針文書、永続化される制作文書では、できる限り肯定形で書く。
- 否定文を使うときは、否定対象が読み手の文脈にあり、否定文全体で指示対象が存在して意味が通ることを確認する。

## 10ページ大学の制作

- プロジェクト方針は `project.md`、シリーズ計画は `series_plan.md`、現在の作業状態は `handoff.md` を参照する。
- 本文制作とレビューは `policies/draft-writing-policy.md` に従う。
- 巻制作、推敲、レビュー反映、HTML/PDF生成、品質確認では repo skill `$ten-page-college` を使う。
- `draft.md`、`preview.html`、PDFを更新したら、`tools/check_volume_quality.py` で画像欠落、PDF陳腐化、本文量を確認する。
